# 真催化日内龙头 vs 纯 Pump

日内候选不是都一样。在给出推荐前,必须先把候选分类。本 skill 服务的是 `+15%~+100%` 当日波动机会,区分真催化、结构性 squeeze 和纯 pump 是给仓位之前的硬门。

## A. 真催化日内龙头 / 财报跳空首日

特征:

- 有可追溯的公司公告、PR、专利、临床、FDA、订单、回购、并购、重大合作、上调指引
- 放量启动,盘前已经站稳承接区
- 涨幅大,但有明确事件驱动
- 板块龙头或同行业先爆,本只票跟涨且自身基本面真实改善

处理:

- 可以列入主攻或备攻
- 仍需检查是不是已经尾声(见下文催化阶段)
- 还要区分硬度:`MOU/LOI` 是软催化,`正式合同/订单/上调指引` 才是硬催化
- 第二/三日延续必须看成交额是否继续放大,价格涨而成交额不跟则不能继续主攻

## B. 结构性 Squeeze

特征:

- 低流通(< 2000 万流通股,有时 < 500 万)
- 有真实或半真实催化(财报、订单、行业事件)
- 短兴趣 ≥ 15% 或 days to cover ≥ 7
- 量能爆炸,但不一定有最硬基本面
- 市场注意力集中,常出现 LULD 停牌恢复

处理:

- 可以做,但仓位通常小于真催化龙头
- 优先做 T,不优先隔夜
- 跌破 VWAP/盘前承接区后立即停止继续做 T

## C. 纯 Pump-and-Dump

特征:

- 没有可信催化
- 只有榜单、社媒、聊天室热度
- 常伴随融资、反拆、历史暴跌
- 拉升和砸盘都极端
- 微盘 Pump 评分(见 [red-flags.md](red-flags.md)) ≥ 9

处理:

- 多数情况只观察
- 若用户明确要赌,也只能给极小仓位和硬止损
- 绝不能伪装成主攻
- 如果它位于当日涨幅榜 Top 1-3、涨幅 `>=80%`、成交额 `>= $10M` 或相对量 `>=50x`,必须进入审计池展示;不能因为它是纯 pump 就静默删除
- 展示时必须写明:`这是防漏展示,不是推荐;只能极小仓触发或观察`

## D. 利好出尽 / 财报反应负面

特征:

- 财报 / 重大公告已经在 24-48 小时前公布
- 历史 4 次财报反应中 ≥ 3 次次日下跌(交叉引用 `earnings_reaction_check.sql`)
- 盘后或盘前先冲高后从高点回撤 `>=12%`,但仍未重新站回 VWAP
- 本日相对量不足前一日的 `30%`

处理:

- 直接降级或排除
- 不允许作为`财报跳空首日`或`财报预判埋伏`的延续主攻
- 若用户在前一日已建仓,切换到[exit-framework.md](exit-framework.md) 的`被套反抽处置`

## 简化判断三问

对每只候选都问 3 个问题:

1. 这波上涨有没有**具体事件**?
2. 这波上涨有没有**持续量能**(当日量比 ≥ 1.5x,延续日量比 ≥ 3x)?
3. 现在参与是不是只能**追最后一棒**?

如果第 1 个回答不了,大概率偏 pump。如果第 3 个回答"是",降级或放弃。

## biotech 追加判断

如果是 biotech,再追加 3 个问题:

1. 事件是不是**已经发生**,而不是未来赌局?
2. 这个事件是不是**足够硬**,比如 Phase 3 / 主要终点 / 死亡率,而不是普通更新?
3. 事件后有没有**二次传播节点**(主流媒体扩散、行业会议 spotlight、机构上调评级),解释为什么还能走第二波?

如果 3 个里有 2 个答不上来,即使涨得很猛,也不要轻易按真催化主攻处理。

## 分时盘口加分判断(日内独有)

日内交易的"是否真催化"还要看分时结构:

- 站上 VWAP 后回踩不破承接 → 加分
- 价格在 VWAP 附近反复横盘但成交额持续放大 → 加分(资金正在吃货)
- 高位横盘但买卖价差扩大、成交断断续续 → 减分(派发特征)
- 尾盘放量砸盘破日内低 → 严重减分
- 当日反复触及 LULD 但每次恢复都向上 → 加分(squeeze 结构)
- 反复触及 LULD 但每次恢复都向下 → 立即放弃
