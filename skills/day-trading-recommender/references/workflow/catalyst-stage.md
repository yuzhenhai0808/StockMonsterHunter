# 催化硬度与日内可交易窗口

本框架用于判断日内候选当前价格还剩多少可交易空间,以及继续上涨/延续需要什么条件。日内不做长期估值,只回答"今天进还来不来得及"和"明天还有没有第二段"。

## 1. 催化硬度分层

把催化分成 6 档,不要把软催化当成硬订单。

| 等级 | 类型 | 例子 | 日内交易含义 |
|---|---|---|---|
| A | 硬合同 / 确定订单 | signed contract, purchase order, definitive agreement, awarded subcontract, revenue guidance raised | 可作日内主攻或财报跳空首日延续主攻 |
| A- | 控制权交易 / 救命融资 | majority stake sale, take-private, fixed-price strategic investment, rescue financing with named buyer | 按事件锚价重估,优先做 T;高位禁追 |
| B | 财报确认 + 业务改善 | revenue beat, profit turn, raised guidance, margin expansion | 财报跳空首日可主攻;第二日必须看延续量能 |
| C | 软合作框架 | MOU, LOI, non-binding proposal, preliminary partnership | 可炒,但只能按事件交易处理;不得按确定收入定价 |
| D | 低确定性叙事 | strategic review, exploring options, vague AI/crypto/quantum wording | 降级,除非盘口结构极强且量能持续 |
| E | 无源热度 | 只有榜单、社媒、聊天室、无可信公告 | 默认 pump,只观察或极小仓 |

处理原则:

- `MOU / LOI / non-binding / preliminary` 是软催化,可推动情绪强化,但不能单独支撑高位重仓
- `definitive agreement / awarded contract / purchase order / guidance raised` 才算硬催化
- 硬催化也可能已经被价格完全透支;催化硬不等于任何位置都能追
- 如果公司公告明确写明"不保证最终协议、融资、交易或预期收益",必须在输出里标注为`软催化未兑现`

### A- 控制权交易额外校验

遇到`按固定每股价注资 / 控股权变更 / 买方成为 CEO 或董事长`的消息,必须单独列出:

- 交易锚价:买方每股价格相对昨收/盘前价的溢价
- 对价质量:现金占比、票据占比、是否 secured note、是否分期
- 交割条件:预计交割日期、是否需要股东/监管/Nasdaq 豁免
- 稀释影响:新增股份占原股本比例,交易后买方持股比例
- 生存价值:是否解决 going concern、违约、债务到期或流动性危机

评分规则:

- `锚价溢价 >= 100%` 且公司公告/8-K + 主流媒体双确认,催化硬度不能低于 `A-`
- 若只有少量现金、主要是票据,或需要豁免/条件较多,只可日内事件交易,不得按长期重估处理
- 若价格已经接近或超过交易锚价,继续上涨必须依赖 squeeze/FOMO,只能按情绪冲刺小仓处理

## 2. 阶段位置分层

每只候选必须判断处于哪一段,直接决定参与方式:

| 阶段 | 常见价格/盘口特征 | 日内交易动作 |
|---|---|---|
| 低位修复 | 刚从底部放量,涨幅相对催化仍不充分 | 可主攻,但仍用止损 |
| 第一轮重估 | 市场开始给新题材/财报改善定价 | 可做回踩或二突 |
| 洗盘后重估 | 重大事件票开盘下杀,随后重新站回 VWAP/开盘价/盘前承接区 | 可从观察升级为回踩做,止损放在 reclaim 失败位 |
| 情绪强化 | 已经连续放量,价格开始提前定价后续利好 | 只做结构,不市价追 |
| 过热逼空 | 当前价远离 VWAP/昨收/催化前价格,需要资金继续接力 | 降仓,只做极小仓二突或放弃 |
| 需要新催化 | 继续上行必须依赖正式协议、订单、指引或二次新闻 | 没新消息只做 T,不做隔夜理由 |
| 催化证伪/退潮 | 跌破关键平台、量能衰减、反抽不过 | 放弃,不补仓 |

判断时至少看:

- `当前价 / 催化前收盘价`
- `当前价 / 当日 VWAP 或盘前 VWAP`
- `当前价 / 20日高点`
- `成交量 / 20日均量`
- 是否已经需要下一条新闻才能解释更高价格

## 3. 各模式偏好阶段

| 模式 | 偏好阶段 | 接受硬度档位 |
|---|---|---|
| 高赔率进攻雷达(默认) | 低位修复 - 第一轮重估 - 洗盘后重估 | A / A- / B;C 需配合放量 |
| 涨幅榜龙头追击 | 第一轮重估 - 情绪强化早段 | A / A- / B |
| 盘前 gap 扫描 | 第一轮重估 - 洗盘后重估 | A / B;C 必须有强承接 |
| 财报跳空首日 | 第一轮重估 | B(财报已确认) |
| 财报预判埋伏 | 低位修复 - 第一轮重估早段 | 间接信号(B-C),赌跳空成 A |
| 强势突破 | 第一轮重估 - 情绪强化早段 | A / B |
| 轧空猎手 | 情绪强化(squeeze 必须有真催化背景) | A / A- / B + short ≥15% |
| 临床事件窗口 | 事件已发生第 1-2 天 | A 临床数据 / B FDA 决议 |
| 微盘爆拉跟风 | 第一轮重估(且总分 ≤ 8) | C 软催化 + squeeze 结构 |
| 事前催化雷达 | 低位修复(等催化) | 待发生事件,只设提醒 |

## 4. 日内 vs 隔夜可执行性

催化阶段决定了它**是否还能持有过夜**:

- 低位修复 / 第一轮重估 / 洗盘后重估 → 仍是事件交易,不持有过夜(本 skill 硬规)
- 情绪强化 / 过热逼空 → 必须当日清仓,不允许任何延续主攻持仓过夜
- 需要新催化 / 催化证伪 → 即使当日仍有反抽,也只能小仓做 T,清仓硬线提前到 14:00 ET

如果用户明确希望持有过夜,必须执行 [exit-framework.md](exit-framework.md) "切换到 swing 的硬规则",改用 long-stock-recommender 重新评估。

## 5. 高位入场纪律

当候选处在`情绪强化`以后,默认禁止直接市价追。

只允许三种处理:

- `回踩做`:回踩关键平台/VWAP/盘前均价后重新放量承接
- `二突做`:高位整理后放量突破,且突破失败的止损点很近
- `放弃`:当前价贴高点、点差宽、无成交连续性、或第一止盈空间小于止损空间

若没有正式硬催化,但已经进入`需要新催化`阶段:

- 可以承认它还能靠 FOMO 冲高
- 但仓位必须降级,输出应写成`情绪冲刺交易`,不是`基本面主攻`
- 第一止盈要更近,盈利后更快减仓

## 6. 输出要求

遇到 MOU/合同/大涨高位票,单票展开必须额外写:

- 催化硬度:`A/A-/B/C/D/E`
- 阶段位置:`低位修复 / 第一轮重估 / 洗盘后重估 / 情绪强化 / 过热逼空 / 需要新催化 / 退潮`
- 继续上涨需要:`继续放量 / 正式协议 / 订单金额 / 指引上调 / 新客户 / 无需新消息但需盘口确认`
- 当前风险:`软催化未兑现 / 估值透支 / 量能衰减 / 红旗融资 / 只剩追高`

总表 `阶段判断` 列若空间允许,可写成组合,例如:

- `情绪强化-C软催化`
- `过热-A硬合同`
- `洗盘后重估-A控制权交易`
- `退潮-红旗`
