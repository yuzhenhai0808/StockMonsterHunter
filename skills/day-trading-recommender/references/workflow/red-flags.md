# 日内红旗清单与 Pump 评分

本文件统一日内的直接拒绝、降级、Pump 评分规则。任一情况命中时,必须按规则降级或排除,并在输出中显式说明。报价/事件认证模板见 [web-cross-check.md](../core/web-cross-check.md);仓位/止损硬线见 [risk-rules.md](../core/risk-rules.md)。

## 直接拒绝

任一命中即直接从候选池剔除,不允许"先标注再保留":

- 过去 7 天内有明显 offering / ATM / toxic financing
- 刚做过 reverse split,且当前暴拉主要靠情绪
- 无法找到可靠催化源
- 已经第 3 天以上连续暴拉
- 纯尾盘突然点火
- biotech 只是赌未来 FDA / 临床 / 数据,事件并未发生
- 用户处于亏损后翻本、满仓、心态失衡状态,且候选属于纯 pump/无硬催化/低流动性彩票
- 当前报价源冲突明显,或只能拿到延迟/过期资料,却需要立即盘中执行
- 微盘 Pump 总分 ≥ 15(典型 pump-and-dump,只观察学习)

## 允许观察,但不推荐

- 盘前已经 `+30%` 到 `+50%`,且没有回踩结构
- 榜首离 VWAP 太远,只能情绪追高
- 前一日已暴拉,第二天没有新催化
- 当日量能不持续,只有价格被点起来
- 前一日/前两日已充分交易的真催化票,当日相对量不足 `3x` 且无法突破前高
- MOU/LOI 软催化叠加 1-2 日涨幅 `>=50%`,且当前价已跌破开盘价/VWAP/盘前承接区之一
- 盘前或盘中先冲高后从高点回撤 `>=12%`,但仍未重新站回 VWAP
- 入场价到建议止损的亏损额会超过用户账户 `2%`,或用户实际仓位已超过账户 `30%`
- 微盘 Pump 总分 9-14(标 🔴,彩票仓 ≤ $50)

## 反拆首日的窄例外

刚做过 reverse split 默认仍是红旗,但如果同时满足以下条件,可以从"直接拒绝"降级为"极小仓战术票":

- 反拆生效首日或次日,流通盘被压缩到极低水平
- 当日成交量相对前 20 日均量放大 `50x+`,最好 `100x+`
- 日内振幅 `30%+`,且不是纯尾盘直线拉升
- 有明确回踩承接或二次突破结构,允许做 T,不允许隔夜
- 没有同日 offering / ATM / toxic financing 同步砸盘

主导偏向写作:`反拆低流通流动性错配`,不是基本面改善。仓位按 [risk-rules.md](../core/risk-rules.md) 降级执行。

## 微盘 Pump 评分表(每个维度 0-2 分)

| 维度 | 0 分 | 1 分 | 2 分 |
|---|---|---|---|
| 市值 | > $5 亿 | $1 亿 - $5 亿 | < $1 亿 |
| 流通股 | > 2000 万 | 500 万 - 2000 万 | < 500 万 |
| 52W 高 / 现价 | < 3 | 3-10 | > 10(有合股历史) |
| 3 日涨幅 | < 30% | 30-80% | > 80% |
| 5 日涨幅 | < 50% | 50-100% | > 100% |
| 年营收 | > $5000 万 | $500 万 - $5000 万 | < $500 万 |
| 盈利 | 正盈利 | 亏损但 FCF 正 | 亏损且 FCF 负 |
| 空头 % | > 15%(轧空结构) | 5-15% | < 5%(纯散户博弈) |
| 7 日内公告 | 财报/业绩 | 订单/合作 | 股东投票/增发/反向拆股 |
| 催化源 | 大型财经媒体 | 行业媒体 | 社交媒体/无明确源 |

### 总分对照

| 总分 | 判定 | 推荐操作 |
|---|---|---|
| 0-4 | 正常股票 | 按其他模式正常处理 |
| 5-8 | 高风险投机 | 彩票仓 ≤ $200,必须标 🟡 警告 |
| 9-14 | 高度疑似 pump | 彩票仓 ≤ $50,标 🔴 警告 |
| 15+ | 典型 pump-and-dump | **不建议任何参与**,只观察学习 |

## 用法纪律

1. 任何 pump 分 ≥ 5 的股,输出时必须把分数和每个维度的红旗明写出来,不能只给操作建议
2. pump 分 ≥ 15 的股,skill 应拒绝推荐,只做教育性分析
3. 用户重复问同一只 pump 股,查 memory 看上次是否已劝阻,是则再次劝阻
4. 不要在 pump 分高的股上叠加"强势突破"或"追涨"模式 —— pump 不是突破

## 输出要求

任一红旗命中,在输出中必须直接说明:

- 命中的红旗是什么
- 是降级还是直接拒绝
- 仍允许参与时的仓位上限(只能写 `极小仓触发 / 只观察 / 放弃`)
- 微盘票必须显示 Pump 总分
