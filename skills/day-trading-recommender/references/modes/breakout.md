# 模式 4:强势突破

**场景**: 大/中市值强势股当天突破 52 周高或 20 日高,放量跟风 | **持仓**: 开盘 1 小时 - 收盘 | **目标**: +5~10% | **胜率**: ~50%

## 触发条件

用户关键词:"突破" / "新高" / "breakout" / "创新高" / "趋势股" / "接着涨"

## 流程

### Step 1 - 本地数据库初筛

调 `long_momentum_candidates(price_buckets="10-100,100-500", limit_per_bucket=5)`,拿到候选后再筛:

- `close_to_20d_high_pct ≥ 97%`(逼近 20日高)
- `volume_ratio ≥ 1.5`
- 市值 ≥ $10 亿(本模式只做中大市值,不碰微盘)

### Step 2 - 52 周新高验证

对每只调 `company_fundamentals`,对比 `close` 与 `fifty_two_week_high`:

- 今日已破 52 周高 → 🟢 突破确认
- 距 52 周高 < 2% → 🟡 即将突破,设条件单在 52W 高 +0.5% 触发
- 距 52 周高 > 5% → 只是 20 日新高,降级

### Step 3 - 量能真实性

调 `symbol_snapshot`,看:
- `volume_ratio` ≥ 2.0 为佳(放量突破)
- `volume_ratio` < 1.5 → 缩量突破,**排除**(假突破概率高)

### Step 4 - ATR 合理性

- ATR14% 在 2-6% 为甜点区(波动够但不疯)
- ATR14% > 10% → 不是本模式,归微盘模式
- ATR14% < 1.5% → 波动太小,日内赚不到 +5%,**排除**

### Step 5 - 财报窗口检查

调 `earnings_dates`,若 `days_until ≤ 3` → **排除**(财报前突破容易被砸盘)

### Step 6 - R/R 计算

标准 -3.5% 硬止损(大市值股 ATR 小):

- 入场上限价 = 52 周高 + 0.5%(突破确认价)
- 硬止损 = 入场价 × 0.965(-3.5%)
- 第一止盈 = 入场价 × 1.05(+5%)
- 第二止盈 = 入场价 × 1.08(+8%)
- R/R = 5/3.5 = **1.4** 🔴 不达标

**如果只能达到 R/R 1.4,必须放宽止盈到 +6%**(R/R = 1.7),或者换标的。

### Step 7 - 输出

按 [../output-contract.md](../core/output-contract.md) 输出模板。

## 输出必须包含

(在 [../output-contract.md](../core/output-contract.md) 输出模板基础上,补充本模式特有字段)

**本模式特有字段**:
- 52 周高价 / 现价距离
- 突破类型(52W 新高 / 20 日新高 / 历史新高)
- 量能比 20日均量的倍数
- 是否有同板块联动

## 需要降级或排除

- 财报 ≤ 3 天 → 排除
- ATR14% < 1.5% → 排除(大盘蓝筹日内无利可图)
- 量能 < 1.5x → 排除(假突破)
- 前一日已经涨 >8% → 已经提前反应,追高风险高
- 当前 RSI14 > 75 → 超买,回吐概率大,降级

## 大市值样板(ARM / TXN 类型)

ARM 4/23 +14.76% 到 52W 高附近就是典型 breakout 模式候选:
- 若用户当日盘前问,会建议: 入场 $232 以下,硬止损 $224,第一止盈 $244(+5%),第二止盈 $251(+8%)
- 14:45 清仓

但 ARM 当时已经接近 52W 高 + 财报 9 天 → 本模式会因**财报窗口**降级为不推荐,改到 earnings-gap-day1 模式在财报后首日再考虑。
