# 日内仓位与纪律

本 skill 默认服务的是高赔率、低胜率、当日波动 `+15%~+100%` 的日内交易。本文件汇总跨模式共享的硬纪律:仓位上限、止损硬线、止盈节点、用户状态守门、概率校准。模式文件可在本规则之上叠加更严的约束,但不允许放松。

## 仓位上限

| 风险等级 | 单笔上限 | 同日并发 |
|---|---|---|
| 主攻:高弹性 + 真实催化 + R/R ≥ 2 | 总资金的 10-20% | |
| 备攻:高弹性 + 结构清晰 + R/R ≥ 1.8 | 总资金的 5-10% | |
| 防守备选:较稳但赔率一般 | 总资金的 10-15% | |
| 极端彩票:微盘 pump + 投机 | 总资金的 1-2% | |
| **同日同时持有日内仓位** | | **≤ 3 只** |

仓位本金用公式反推:`建议本金 = 账户风险额 / 止损距离`。例:`$5,000` 账户、本单愿亏 `$50`、止损距离 `5%`,建议本金 ≈ `$1,000`。

**禁止满仓**:本 skill 不允许单票本金超过账户 `30%`;超过 `50%` 必须先降仓再讨论交易计划。**主攻不等于重仓** —— `主攻` 只是候选优先级,不是本金仓位;输出必须同时写`建议仓位上限`和`最大亏损`。

## 用户状态守门(任何模式之前判断)

| 用户状态 | 限制 |
|---|---|
| 当日已亏 ≥ 2 笔 | 剩余交易日禁止再开仓 |
| 当周连亏 5 笔 | 停交易 3 天,回看每一笔的止损执行 |
| 当日已超过最大亏损预算(建议 5%/日) | 停手 |
| 睡眠 < 5 小时 / 刚与家人争吵 / 情绪化 / FOMO | 不开仓 |
| 单笔账户风险额已使用 ≥ $50(对 $5k 账户) | 后续同票只能用利润仓 |
| 单轮盈利达到账户 5%+ | 默认建议停止同票交易;若继续,只能用已实现利润 10%-20% 做第二轮最大亏损 |
| 用户对该票本金 >30% | 必须先给"账户风险情景概率表"再讨论上涨 |

被触发止损后强制等 30 分钟再决定下一笔(防报复性交易)。

## 时间纪律

- 默认只做日内,默认不过夜
- `14:45 ET` 强制清仓(无论盈亏)
- 最佳建仓窗口: `09:45 ET - 11:00 ET`
- 尾盘点火默认不参与
- 09:30-09:45 ET 默认观察;只允许盘前已挂的条件单触发
- 11:00-13:30 ET 若未形成第二波,优先减仓
- 13:30-14:30 ET 只管理仓位,少开新仓

详细时段动作见 [output-contract.md](output-contract.md) "时间上下文自适应"。

## 止损硬线

| 股票类型 | ATR14% | 默认硬止损 | 第一止盈 | 第二止盈 | R/R |
|---|---|---|---|---|---|
| 低波动突破 | 2-4% | -3.5% | +5% 到 +6% | +8% | 1.4~2.3 |
| 财报跳空首日 | 4-8% | -4% | +8% | +15% | 2.0~3.8 |
| 盘前 gap 中盘 | 5-10% | -4.5% | +10% | +15% | 2.2~3.3 |
| 轧空候选 | 5-10% | -5% | +10% | +18% | 2.0~3.6 |
| 高弹性主攻 | 8-15% | -5% | +12% | +20% | 2.4~4.0 |
| 微盘 pump | 10-20% | -5% | +8% | +15% | 1.6~3.0(仓位极小) |

公共硬约束:

- 默认硬止损 `-3% 到 -5%`,不过 `-5%`,否则改用极端彩票仓位
- 第一止盈 `+8%` 起,保证 R/R `≥ 1.5`(低于 1.5 直接放弃)
- 时间止损 `14:45 ET`,无论盈亏必平仓
- 止损单**必须在买入时同时挂**(盘前能挂就挂)
- 止损被触发后,**同一只股当日不再买**(除非有重大新信号)
- 跌到止损位前**不得加仓**(摊平是日内最大忌)

止损价考虑技术支撑:

```
硬止损价 = max(入场价 × (1 - 止损%), 今日 VWAP, 盘前最低价)
```

技术位破了说明买盘吃不住,动能已失;百分比止损是数学底线,技术位是更早的预警。

## 止盈纪律

- 第一止盈触发时:减仓 `50%`,剩余仓位的止损抬到入场价
- 第二止盈触发时:全部清仓,不贪
- 第二止盈未达到 → `14:45 ET` 无条件平仓
- 已经完成一轮 `8%-20%` 做 T 后,默认至少兑现一半以上
- 单轮盈利达到账户 `1.5%-2%` 后,后续同票只能用同等或更小风险继续,不能利润上头加码
- 单轮盈利达到账户 `5%+` 后,默认建议停止同票交易;若继续,只能用已实现利润的 `10%-20%` 作为第二轮最大亏损

## 概率校准(必做)

凡是输出 `第一止盈 / 第二止盈 / 回本价 / 突破价 / 下破价` 等价格预测,必须同时给出概率区间。

- **概率必须绑定证据等级**:每个概率区间旁边必须写 `证据等级 A/B/C/D` 和关键依据。`A/B` 可以给入场后先到止盈/止损概率;`C` 只能给`条件触发后的粗略情景概率`;`D` 不允许给概率,只能写`报价不足,不估概率`
- **禁止无依据推荐**:若某只票没有可复核的最新价、报价时间和来源,不能出现在正常推荐 Top 1-3;最多进入`观察/待报价复核`或`极端彩票审计池`。不得用本地日线涨幅、新闻热度或历史强势替代当前报价证据
- 日内/盘前票默认给`入场后先到第一止盈概率`和`入场后先到止损概率`;持仓处置问题给 `反抽触达概率` 和 `跌破风险概率`
- 概率必须用区间表达,例如 `35%-45%`,不要写成精确单点
- 估算依据至少引用 3 项: `ATR14% / 当日或盘前量比 / 距 VWAP 或关键高低点 / 催化硬度 / 是否第 2-3 天 / 当前报价是否可验证 / 大盘与板块方向`
- 概率必须以`计划入场价`为起点,衡量价格先触及`第一止盈`还是先触及`硬止损`;不要用"当天可能触达目标"替代交易触发后的路径概率
- 若价格已经越过入场上限,必须重算为`回踩做/二突做`的新概率,否则降级为`只观察`
- 若缺少实时 VWAP、盘口或夜盘有效成交,必须写明"概率基于最近完整日线 + 可见 Web 报价,置信度降低",不得把概率写得过窄
- 默认概率分档: `>=60%` 高概率但通常肉薄, `40%-60%` 中性可交易, `20%-40%` 赔率型, `<20%` 彩票/尾部情景

## 流动性门(硬线)

- 默认 `20 日均成交额 ≥ $5,000,000`
- 当日盘前/盘中成交量比 `≥ 1.5x`
- **高弹性例外**:squeeze / 龙头追击 / 高赔率进攻雷达主攻池可放宽到 `≥ $2,000,000`,标"高波动,滑点更大"
- **财报预判例外**:`财报预判埋伏` 候选若满足"历史财报反应、指引/一致预期修正、板块传导、无资本事件红旗" 4 项中至少 3 项,可放宽为 `≥ $10,000,000`,且不强制要求当日量比提前放大
- **微盘模式例外**:允许 `≥ $500,000`,但必须标"低流动性,仓位受限"

不满足直接排除,不要"可惜了"式挽留。

## R/R 计算简例

案例 A:高弹性主攻 $20 入场,硬止损 $19.00 (-5%),第一止盈 $22.40 (+12%) → R/R = 12/5 = 2.4 ✅,可主攻

案例 B:TXN $279 入场(财报跳空首日),硬止损 $267.84 (-4%),第一止盈 $301.32 (+8%) → R/R = 8/4 = 2.0 ✅,适合本框架

案例 C:ARM $234 入场(低波动突破),硬止损 $225.81 (-3.5%),第一止盈 $245.70 (+5%) → R/R = 5/3.5 = 1.43 🔴 不达标,必须放宽止盈到 +6% 否则放弃

## 不适合日内交易的人

诚实提醒以下用户**不应该**做日内:

- 需要工作时间处理其他事务(日内要求盯盘)
- 本金 < $5000(佣金+滑点占比太高)
- 不能承受 -20% 月度回撤
- 交易动机是"翻本"或"快速致富"

## 与 swing 交易的切换规则

如果用户在日内模式下找到了**不想平仓的股**(比如 TXN 财报跳空后想拿 1-2 周):

1. **平掉日内仓位**(按本 skill 规则)
2. 重新用 long-stock-recommender 评估是否值得 swing 持有
3. 若决定 swing,用**新的资金**建立 swing 仓位,止损和目标按 swing 框架设

**不要让日内仓位变成 swing** —— 这是"你拿不住的票强行拿住"的典型错误,多数以更大亏损收场。

## 记录纪律

每笔交易记录:入场时间/价位、止损/止盈价位、实际退出时间/价位、盈亏、退出原因、事后复盘是否严格执行了止损。
