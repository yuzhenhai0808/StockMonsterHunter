# 风险与仓位规则（所有模式共享）

本文件汇总各模式共享的硬纪律：仓位上限、止损硬线、降级与排除条件、用户状态守门。模式文件可在本规则之上叠加更严的约束，但不允许放松。

## 仓位上限

| 场景 | 单票上限 | 组合上限 |
|---|---|---|
| 追涨（默认） | 5-8% | 30-40% |
| 超跌反弹 | 3-5% | 15% |
| 回调买入 | 5-8% | 30% |
| 价值挖坑 | 5-10%（按市值档） | 单档 ≤30%，合计 ≤50% |
| **财报跳空（模式 5）** | **≤2.5%** | **≤15%（必须 ≥5 只、最好 8 只）** |
| 叠加信号（胜率最高）| 4-5%，不超过 8% | ≤20% |

财报跳空模式硬约束：
- 单票 ≤2.5%（不允许情绪化加仓）
- 必须同时埋伏 ≥5 只，单押失败不适用
- 8 只组合 = 20% 仓位上限，**不允许放大**

## 用户状态守门（在任何模式之前判断）

| 用户状态 | 限制 |
|---|---|
| 刚亏完 / 焦虑 / 急着翻身 | **拒绝推荐模式 5**；劝阻并改推稳健方案 |
| 账户总仓位 > 70% | 不允许进模式 5，先减仓 |
| 账户总仓位 > 80% | 模式 6 也不允许；先减仓 |
| 单票 2.5% < $500 | 模式 5 不适用（账户太小，样本不够分散）|
| 用户对该票本金 >30% | 必须先给"账户风险情景概率表"再讨论上涨 |

## 大盘环境

模式 5 / 6 额外要求：
- VIX < 25
- SPY 20 日 > -5%

不满足 → 即使叠加信号也会被拖累，改推稳健仓。

## 止损硬线

| 模式 | 止损规则 |
|---|---|
| 追涨 | MAX(`atr_stop = close - 1.15*ATR14`, `trend_stop = MA10*0.97`, `day_low_stop = low*0.98`)；硬约束 `close*0.82 ≤ stop ≤ close*0.94` |
| 超跌反弹 | 20 日低点 * 0.97 立即止损 |
| 回调买入 | MAX(MA20*0.99, MA50*0.97)；破 MA20 即止损 |
| 价值挖坑 | **不给硬止损**，给分批建仓区间（当前 -5% 到 +2%）与"持仓周期 + 退出信号" |
| 财报跳空 | 跳空后次日开盘 -5% 全清；硬上限 -15% |
| 叠加信号 | 按主模式止损执行，不混用 |

## 止盈规则

| 模式 | 止盈 |
|---|---|
| 追涨 | `MAX(high_20d*1.08, close + 1.8*risk, close + 1.35*ATR14)`，最低 R:R 1.6 |
| 超跌反弹 | 反弹到 MA20 即减仓 |
| 回调买入 | 20 日前高 * 1.05（结构性突破前高的预期） |
| 价值挖坑 | 估值修复到 P/E 20 或业绩出坑 / 行业复苏 |
| 财报跳空 | 任一票财报后 3 天内 +30% → 锁定 2/3 利润；不 hold 过多于 1 次财报 |
| 叠加 A | 财报后 +30% 减 2/3，-10% 全清 |
| 叠加 B | 财报后跳空 +15% 锁 2/3；否则目标 20 日前高 * 1.05 |
| 叠加 C | 持仓 3-6 月，按价值模式退出信号 |

## 通用降级 / 排除条件

任一命中即降级（标注+减仓）或排除（不推荐）：

- 当前价格已远高于脚本买入区间（追高）
- 硬催化已被多日暴涨完全兑现（20 日涨幅 >150%、RSI >85 且当日冲高回落）→ 列为复盘/观察，不能开新主推
- 只因为股价低进入候选，但成交额、趋势质量都不够
- 近期 / 潜在增发、ATM、SEPA、可转债、权证、反向拆股、Nasdaq 合规风险 → 直接排除
- 生科股临近关键临床读数（30 天内）→ 二元风险，不作为趋势推荐
- 本地数据库价格与外部行情明显不一致 → 必须以 Web 为准重算
- 回调原因是公司层面利空（WebSearch 近 5 天新闻确认）
- 财报日 ≤7 天（不开新仓）
- 跌破 MA50 超过 3%（趋势破坏）
- Float < 20M 或 Short > 20% → 标注 squeeze 属性，不作主推
- 微盘股纯技术异动无新闻支撑 → 默认视为操纵
- 价值模式：审计师辞职 / SEC 调查 / 高管集体抛售 / 一次性利得撑利润 → 排除（不可恢复）

## 价值模式特别规则

- 持仓 ≥3 个月，不适合短线
- 必须给"持仓周期与退出信号"而不是硬止损
- 退出信号：估值修复到 P/E 20 / 业绩出坑 / 行业复苏 / 突破历史阻力
