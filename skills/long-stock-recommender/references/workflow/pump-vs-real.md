# 真业绩拐点 vs 概念炒作 vs 利好出尽

做多候选不是都一样。在给出推荐前,必须先把候选分类。本 skill 侧重 swing(1-6 周),核心问题不是"这只票今天能不能涨",而是"这次到底靠什么赚钱、能撑多久、是不是已经被定价完了"。

## A. 真业绩拐点 / 硬催化主线

特征:

- 有可追溯的公司公告、订单、合同、上调指引、产能落地、并购、回购
- 财报后基本面真实改善:营收/利润/现金流连续向好
- 板块景气先行抬升,且本只票在板块内仍有相对估值空间
- 放量启动,且后续仍有机构资金确认

处理:

- 可以列入主推或备选
- 仍需检查是不是已经尾声(见 [catalyst-stage-framework.md](catalyst-stage.md) 阶段 4-5)
- 区分硬度等级:`MOU/LOI` 是软催化,`正式合同/订单/上调指引` 才是硬催化
- 持仓周期可拉长到 3-6 周,允许第一目标后只减半

## B. 板块情绪 / 主题炒作

特征:

- 受行政令、政策利好、行业 ETF 资金推动
- 板块内龙头先爆,本只票跟涨,但自身基本面未明显变化
- 流动性有一定支撑,但叙事高度依赖"下一个利好"
- 通常对应 [catalyst-stage-framework.md](catalyst-stage.md) 中的 **C 档软催化** 或 **D 档低确定性叙事**

处理:

- 可作为备选,但仓位减半
- 输出必须显式标"板块情绪驱动,回归均值风险"
- 第一目标更近,达到后立即兑现一半以上
- 只要板块龙头放量见顶或政策预期被消化,立即降级或排除
- 不允许作为价值模式或叠加模式 C 的主导偏向

## C. 利好出尽型 / 财报前已被定价

特征:

- 催化剂在财报前已经被市场充分买入(分析师上调、空头加码、社媒热议)
- 历史 4 次财报中 ≤ 1 次次日上涨,或 EPS 超预期仍跌 ≥ 2 次
- 财报日反而成为获利了结窗口

处理:

- **直接排除**(无论催化剂多强)
- AEVA 2026-04-27 教训:Nvidia 合作 + 25% 空头 + Strong Buy 在财报前看似完美,但 4 次财报 3 次跌,属典型利好出尽
- TSEM 2026-04-27 教训:回调到 MA20 结构完美,但历史 4/4 跌 + 管理层亲口下调指引,首推前三连校验全红
- 这类候选不允许靠"主导偏向漂亮"绕过历史财报反应一票否决

## D. 纯技术异动 / 庄家操纵

特征:

- 微盘股(市值 < $1 亿),无可信新闻支撑
- 只有 K 线漂亮、放量、突破前高,但找不到对应的公司公告或行业催化
- 常伴随定增、权证、反向拆股或可转债历史

处理:

- 默认拒绝推荐
- 若用户明确点名,只能列为"观察清单",标"举证责任反转"
- 绝不能伪装成"基本面好的小盘成长股"

## 简化判断三问

对每只候选先问 3 个问题:

1. 这次上涨有没有**具体事件 / 业绩拐点**?
2. 这个事件 / 拐点是**真实改变了未来 1-3 年现金流预期**,还是**只是情绪 / 板块联动**?
3. 现在参与是不是只能**追最后一棒**(已进入阶段 4-5)?

如果第 1 题回答不了,或第 3 题答案是"是",大概率应放弃或降级。

## 财报相关追加判断

如果催化是财报或财报前埋伏,再追加 4 个问题:

1. 历史 4 次财报次日反应如何?(≤1/4 上涨 → 排除)
2. 管理层是否已发布下季度指引?指引中值 vs 上季度实际谁高?
3. 公司是否在历史财报后出现"先深砸再修复"的路径?(≥ 2 次 → 降级)
4. 当前估值与"如果财报符合预期"的隐含估值有没有显著预期差?

任何一题答不上来或方向不利,即使其他条件漂亮,也不要作为首推。

## 价值模式追加判断

价值模式追加 3 个问题:

1. 营收/营业利润/经营性现金流近 4 季度是否同向上升?
2. trailing P/E 是否显著低于 sector 中位数?P/B 是否合理?
3. 公司是否存在会计问题、审计师辞职、CFO 离职、SEC 调查?

任一题不利则降级或排除。
