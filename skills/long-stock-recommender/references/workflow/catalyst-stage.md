# 催化剂硬度与阶段框架（所有模式共享）

本文件定义催化剂的**硬度等级**与**阶段成熟度**，所有模式在判断"主导偏向是否成立 / 是否值得重仓 / 是否已经被定价"时引用本框架。

## 一、催化剂硬度（A / A- / B / C / D 五档）

| 档 | 含义 | 例子 | 处理 |
|---|---|---|---|
| **A** | 硬合约、订单、PO、确定性协议、上调指引 | MRAM 4/30 $40M、2.5 年 Mil-Aero MRAM 协议（约 TTM revenue 70%+）；EPS+营收+指引三杀超预期 | 可作主导偏向；可作重仓主推 |
| **A-** | 多数股权 / 私有化 / 战略投资按固定每股价 | 大客户长期合作 + 板块景气共振；同行业巨头先爆财报 | 可作主导偏向；仓位略低于 A |
| **B** | 业绩超预期 + 业务真实改善 | Q1 EPS beat + 营收同比加速 + 利润扩张 | 可作主导偏向；按各模式仓位 |
| **C** | MOU、LOI、框架协议、非约束性合作、新专利、临床中期里程碑 | 合作意向书；FDA Phase II 数据；行政令利好 | 可交易但不可按"已确认收入"定价；仓位减半 |
| **D** | 模糊战略评估 / 网络热度 / 板块情绪 / Reddit & Twitter 热议 | 无明确公告，仅社交热度 | 标"情绪驱动，回归均值风险"；不作主推 |
| **E** | 无任何新闻，纯技术暴涨 | 微盘股 K 线漂亮但无公告 | **微盘股默认视为操纵**，举证责任反转 |

**纪律**：
- 不允许把 C 档当作 A 档定价
- 模式 5 财报跳空中，"间接信号"（分析师上调、同行爆发、营收加速）属 B-C 档，是猜概率不是确认
- 对单一 PR 加分有限，必须前置铺垫（如 MRAM 先 4/8 Microchip on-shore 协议，再 4/30 Mil-Aero 合同）

## 二、阶段成熟度（六阶段）

判断当前股价处在催化剂作用的哪个阶段，决定参与方式：

| 阶段 | 特征 | 操作建议 |
|---|---|---|
| 1. 低位修复 | 长期低迷后催化剂初现，量能初步放大 | 可建底仓 |
| 2. 首次重估 | 突破前 20 日高 + ≥3x 量；机构资金确认 | 主推/加仓窗口 |
| 3. 洗盘后再估 | 第一波后回踩 MA10/MA20，量缩 | 回调买入主战场 |
| 4. 情绪强化 | 连续 3-5 天放量上涨，RSI 70+，分析师跟进上调 | **不再追**；持有者考虑减半 |
| 5. 过热挤压 | 20 日涨幅 >150%、RSI >85、当日冲高回落、放巨量 | 列复盘/观察，不开新主推 |
| 6. 需新催化 / 已被证伪 | 价格停滞或回吐 30%+，无新增催化 | 退出，等下一波 |

**纪律**：
- 阶段 4-5 进入禁追区；阶段 5 之后即使强势也只能极短线处理
- "突破点"在阶段 1-2 才有意义；阶段 4+ 出现的"突破"通常是末段诱多

## 三、与各模式的对应

| 模式 | 偏好阶段 | 接受硬度档位 |
|---|---|---|
| 追涨 | 阶段 2-3 | A / A- / B；C 需配合放量 |
| 超跌反弹 | 阶段 6 后 → 进入阶段 1 | 不强制硬度；只要跌幅是"无新增利空" |
| 回调买入 | 阶段 3 | A / A- / B 主导偏向不变 |
| 价值挖坑 | 阶段 1 | B 业绩拐点 + 估值；不依赖单一催化 |
| 财报跳空 | 阶段 1 末 - 阶段 2 初 | 间接信号（B-C），赌跳空成 A |
| 叠加 A | 阶段 2 + 财报窗口 | A 概率最高（MXL 样板） |

## 四、催化剂真实性验证（micro-cap 必做）

| 类型 | 持续性 | 处理 |
|---|---|---|
| 真实订单 / 合同 / 业绩超预期 | 高 | 正面佐证 |
| 新专利 / FDA 里程碑 / 临床 II-III 数据 | 中-高 | 正面佐证，需看商业化路径 |
| 行政令 / 政策利好 / 板块情绪 | 低 | 标"板块情绪驱动，回归均值风险" |
| 网络热度 / Meme | 极低 | 标"情绪驱动，非趋势交易" |
| **无任何新闻** | 极低 | **微盘股默认视为操纵** |
