# 妖股红旗清单

本文件只负责**枚举红旗及其降级/拒绝动作**。其他职责不在本文件:

- 仓位上限/账户风险刹车的具体阈值 → [../core/risk-rules.md](../core/risk-rules.md)
- MOU/LOI 软催化的阶段判断 → [catalyst-stage.md](catalyst-stage.md)
- 真催化 / squeeze / pump 分类 → [pump-vs-real.md](pump-vs-real.md)

以下情况默认降级或拒绝:

## 直接拒绝

- 过去 7 天内有明显 offering / ATM / toxic financing
- 刚做过 reverse split,且当前暴拉主要靠情绪
- 无法找到可靠催化源
- 已经第 3 天以上连续暴拉
- 纯尾盘突然点火
- biotech 只是赌未来 FDA / 临床 / 数据,事件并未发生
- 用户处于亏损后翻本、满仓、心态失衡状态,且候选属于纯 pump/无硬催化/低流动性彩票票
- 当前报价源冲突明显,或只能拿到延迟/过期资料,却需要立即盘中执行
- SPAC/壳股/停牌恢复票无法确认赎回压缩、停牌恢复结构或真实催化,却只靠低流通叙事拉升
- 低价票同时命中`负股东权益/现金极低/高成本融资/近期稀释风险`中 3 项,且实时量价没有 A 级确认
- AI 贴标签票: 48 小时内没有公司 PR/IR、SEC 8-K/6-K 或主流财经源可追溯催化,只有社媒/榜单/旧闻把公司描述成 AI 概念,且无法说明金额、客户、部署节点或收入路径

## 允许观察,但不推荐

- 盘前已经 `+30%` 到 `+50%`,且没有回踩结构;若同时满足催化可追溯、当前成交额足够、点差可控、VWAP/整数位/平台承接、第一目标仍有 `6%-12%` 空间,可降级为`小仓做 T`,否则只观察
- 榜首离 VWAP 太远,只能情绪追高
- 前一日已暴拉,第二天没有新催化
- 当日量能不持续,只有价格被点起来
- biotech 只有会议展示/口头报告,但核心数据其实是旧闻
- 前一日/前两日已充分交易的真催化票,当日相对量不足 `3x` 且无法突破前高
- 软催化(MOU/LOI/探索性合作/`需要新催化`阶段/高位透支等)的具体阶段位置判定与高位透支阈值: 按 [catalyst-stage.md §阶段位置分层](catalyst-stage.md) 与 §A-控制权交易额外校验执行,本文件不另定阈值
- 盘前或盘中先冲高后从高点回撤 `>=12%`,但仍未重新站回 VWAP
- 仓位/止损红线触发(账户风险 / 单票本金占比超限): 按 [../core/risk-rules.md](../core/risk-rules.md) 的硬上限处理,本文件不另定阈值
- 开盘 5 分钟内垂直拉升、无回踩承接、无法定义止损或 bid-ask 明显扩张
- AI 软催化票: AI development agreement、MOU/LOI、AI 顾问任命、AI data center 意向、AI pivot 或探索性合作,但商业部署和收入路径未确认;若同时低流通和成交额跃迁,最多`极小仓触发/等二突`,不得按基本面重估建仓

## 低价财务红旗加权 (NXXT 型)

低价高弹性票允许研究,但财务红旗会限制仓位等级。以下每项 +1:

| 红旗 | 判定 |
|---|---|
| 负股东权益 | `stockholders_equity < 0` 或 profile book value 为负 |
| 现金极低 | `cash < $1M` 或现金不足覆盖一个季度经营现金流 |
| 高成本融资 | 30 天内出现高利率债务、ELOC、toxic financing、全资产/CEO担保融资等 |
| 近期稀释风险 | 30 天内 S-1/F-1/S-3/ATM/offering/equity line/注册生效 |
| 持续亏损 | 最近 4 季度多数为净亏损或 operating margin 明显为负 |

处理:

- 红旗分 `0-1`: 可按正常高弹性候选继续复核。
- 红旗分 `2`: 最高只能`备攻/极小仓触发`,账户风险默认 `<=0.5%`。
- 红旗分 `>=3`: 最高只能`观察/等reclaim`;除非实时 quote A 级、当前成交额通过主攻硬门、且价格重新站回 VWAP/开盘价/盘前承接区,否则不得写主攻/备攻。
- 若同时出现用户亏损后翻本、满仓或补仓诉求,红旗分 `>=2` 的低价票直接禁止作为回本主路径。

## 反拆首日的窄例外

刚做过 reverse split 默认仍是红旗,但如果同时满足以下条件,可以从"直接拒绝"降级为"极小仓战术票":

- 反拆生效首日或次日,流通盘被压缩到极低水平
- 当日成交量相对前 20 日均量放大 `50x+`,最好 `100x+`
- 日内振幅 `30%+`,且不是纯尾盘直线拉升
- 有明确回踩承接或二次突破结构,允许做 T,不允许隔夜重仓
- 没有同日 offering / ATM / toxic financing 同步砸盘

这种票的主导偏向写作: `反拆低流通流动性错配`,不是基本面改善。仓位和账户风险按 [../core/risk-rules.md](../core/risk-rules.md) 降级执行。

阈值边界: `50x+` 是降级为"极小仓战术票"的最低门槛(本节);`>=100x` 是同步进入极端彩票池审计的标准(按 [../modes/extreme-lottery-leaderboard.md §入池条件](../modes/extreme-lottery-leaderboard.md))。两条规则可以同时生效,本节负责"是否允许做",彩票池负责"是否需要展示"。

## 输出要求

一旦命中红旗,必须在输出里直接说明:

- 命中的红旗是什么
- 它是降级还是直接拒绝
- 如果仍允许参与,仓位上限是多少(具体数字按 [../core/risk-rules.md](../core/risk-rules.md))
