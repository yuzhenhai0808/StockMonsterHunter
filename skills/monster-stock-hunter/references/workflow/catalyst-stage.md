# 催化硬度与阶段位置框架

这个框架用于处理已经大涨的事件票,尤其是用户问:

- `现在高位还能不能入`
- `MOU 是什么概念`
- `没有正式协议还能不能继续涨`
- `这只票属于第几段`
- `涨到这里是否还合理`

目标不是给长期估值,而是判断**当前价格还剩多少可交易空间**,以及继续上涨需要什么新条件。

## 1. 催化硬度分层

先把催化分成 5 档,不要把软催化当成硬订单。

| 等级 | 类型 | 例子 | 交易含义 |
|---|---|---|---|
| A | 硬合同 / 确定订单 | signed contract, purchase order, definitive agreement, awarded subcontract, revenue guidance raised | 可按真催化妖股处理,但仍看是否已过热 |
| A- | 控制权交易 / 救命融资 | majority stake sale, take-private proposal, strategic investment at fixed per-share price, rescue financing with named buyer | 可按事件锚价重估处理,但必须拆分现金比例、票据/或有条件、稀释和成交风险 |
| B | 财报确认 + 业务改善 | revenue beat, profit turn, raised guidance, margin expansion | 支撑第一轮重估,但不一定支撑连续情绪冲刺 |
| C | 软合作框架 | MOU, LOI, non-binding proposal, preliminary partnership | 可以炒,但只能按事件交易处理;不得按确定收入估值 |
| D | 低确定性叙事 | strategic review, exploring options, vague AI/crypto/quantum wording | 降级,除非盘口结构极强且量能持续 |
| E | 无源热度 | 只有榜单、社媒、聊天室、无可信公告 | 默认 pump,只观察或极小仓 |

处理原则:

- `MOU / LOI / non-binding / preliminary` 是软催化,可以推动`情绪强化`,但不能单独支撑`高位重仓`。
- FDA `may support / pathway / Q-sub minutes / meeting minutes` 属于路径型利好,最高按 B-/C+ 处理;它证明监管路径存在,不等于 approval、PMA/NDA/BLA 已提交、获批或商业化收入。若同时有现金 runway、Nasdaq 合规、融资需求或开盘承接失败,默认降为`观察 / 等reclaim`,不得主推。
- `AI development agreement / AI pilot / AI sales agent / reverse merger / AI pivot` 若缺少明确收入确认、交割条件、客户部署或指引上调,按 C/D 处理;即使盘中暴涨,也只能入极端彩票审计或极小仓触发。
- `reverse split 后低流通动量` 不是基本面催化;只有 A/B quote、当前成交额通过硬门、VWAP/reclaim 或二突成立时,才允许极小仓战术。
- `definitive agreement / awarded contract / purchase order / guidance raised` 才算硬催化。
- `majority stake / strategic investment / acquisition proposal` 若有具名买方、每股价格、交易规模、预计交割时间和公司公告/8-K,按 `A-控制权交易` 处理。不要因为公司基本面差就自动降为彩票;这类票赚的是事件锚价重估和流动性挤压。
- 硬催化也可能已经被价格完全透支;催化硬不等于任何位置都能追。
- 如果公司公告明确写明“不保证最终协议、融资、交易或预期收益”,必须在输出里标注为`软催化未兑现`。

### A-控制权交易额外校验

遇到类似 `按固定每股价注资 / 控股权变更 / 买方成为 CEO 或董事长` 的消息,必须单独列出:

- 交易锚价:买方每股价格相对昨收/盘前价的溢价
- 对价质量:现金占比、票据占比、是否 secured note、是否分期
- 交割条件:预计交割日期、是否需要股东/监管/Nasdaq 豁免
- 稀释影响:新增股份占原股本比例,交易后买方持股比例
- 生存价值:是否解决 going concern、违约、债务到期或流动性危机

评分规则:

- `锚价溢价 >= 100%` 且 `公司公告/8-K + 主流媒体` 双确认,催化硬度不能低于 `A-`
- 若只有少量现金、主要是票据,或需要豁免/条件较多,不得给长期投资结论,但仍可给日内事件交易
- 如果盘中先杀跌但未跌破催化前收盘,随后重新站回 VWAP/开盘价,这是`洗盘后重估`结构,不应自动降级为失败
- 若价格已经接近或超过交易锚价,继续上涨必须依赖 squeeze/FOMO,只能按情绪冲刺小仓处理

## 2. 阶段位置分层

每只候选必须判断处于哪一段:

| 阶段 | 常见价格/盘口特征 | 交易动作 |
|---|---|---|
| 低位修复 | 刚从底部放量,涨幅相对催化仍不充分 | 可主攻,但仍用止损 |
| 第一轮重估 | 市场开始给新题材/财报改善定价 | 可做回踩或二突 |
| 洗盘后重估 | 重大事件票开盘下杀,随后重新站回 VWAP/开盘价/盘前承接区 | 可从观察升级为回踩做,止损放在 reclaim 失败位 |
| 情绪强化 | 已经连续放量,价格开始提前定价后续利好 | 只做结构,不市价追 |
| 过热逼空 | 当前价远离 VWAP/昨收/催化前价格,需要资金继续接力 | 降仓,只做极小仓二突或放弃 |
| 需要新催化 | 继续上行必须依赖正式协议、订单、指引或二次新闻 | 没新消息只做 T,不做隔夜理由 |
| 催化证伪/退潮 | 跌破关键平台、量能衰减、反抽不过 | 放弃,不补仓 |

判断时至少看:

- `当前价 / 催化前收盘价`
- `当前价 / 当日 VWAP 或盘前 VWAP`
- `当前价 / 20日高点`
- `当前市值 / 年化收入` 或 `EV / EBITDA`(若可得)
- `成交量 / 20日均量`
- 是否已经需要下一条新闻才能解释更高价格

软催化高位透支阈值(命中任一即归入 `需要新催化` 或 `过热逼空`,按表格交易动作执行):

- MOU/LOI/框架合作叠加 1-2 日涨幅 `>=50%`,且当前价已跌破开盘价/VWAP/盘前承接区之一
- 已经进入`需要新催化`阶段,但没有正式协议、订单金额、客户落地、指引上调或二次新闻,只能靠 FOMO 继续推高
- 软催化被市场按确定订单或确定收购大幅重定价,但合同金额、交割条件、收入路径仍缺失

## 3. 事前可推演性检查

为了避免事后解释,对每只高位票都问:

1. 如果在起爆初期看到同样信息,能否事前推导出一段合理目标区间?
2. 这个目标区间来自基本面修复、事件驱动,还是纯资金情绪?
3. 当前价格是否已经超过当时可推导的第一段目标?
4. 若要继续上行,需要什么新增信息?

输出时把结论写成概率和条件,不要写成确定目标价。

示例表达:

- `3 美元看 5 美元是事件驱动可推演,但不是基本面确定目标。`
- `6.3 美元再看 8 美元,已经需要更强资金情绪或新催化。`
- `35 美元以上继续追,不是买合同本身,而是在赌订单后第二波定价。`

## 4. 高位入场纪律

当候选处在`情绪强化`以后,默认禁止直接市价追。

只允许三种处理:

- `回踩做`: 回踩关键平台/VWAP/盘前均价后重新放量承接。
- `二突做`: 高位整理后放量突破,且突破失败的止损点很近。
- `放弃`: 当前价贴高点、点差宽、无成交连续性、或者第一止盈空间小于止损空间。

若没有正式硬催化,但已经进入`需要新催化`阶段:

- 可以承认它还能靠 FOMO 冲高。
- 但仓位必须降级,输出应写成`情绪冲刺交易`,不是`基本面主攻`。
- 第一止盈要更近,盈利后更快减仓。

## 5. 输出要求

遇到 MOU/合同/大涨高位票,单票展开必须额外写:

- 催化硬度: `A/A-/B/C/D/E`
- 阶段位置: `低位修复 / 第一轮重估 / 情绪强化 / 过热逼空 / 需要新催化 / 退潮`
- 继续上涨需要: `继续放量 / 正式协议 / 订单金额 / 指引上调 / 新客户 / 无需新消息但需盘口确认`
- 当前风险: `软催化未兑现 / 估值透支 / 量能衰减 / 红旗融资 / 只剩追高`

表格中若空间允许,把`阶段判断`写成组合,例如:

- `情绪强化-C软催化`
- `过热-A硬合同`
- `洗盘后重估-A控制权交易`
- `退潮-红旗`
